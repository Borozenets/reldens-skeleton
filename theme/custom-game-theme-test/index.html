Інструкції

  • Керування стрілками або клавішами W-A-S-D.
  • Клацніть лівою кнопкою, щоб йти вперед. Щоб змінити сцену, підійдіть до точки переходу (наприклад, міст чи двері будинку).
  • Клацніть правою кнопкою або натисніть TAB для вибору цілі поруч.
  • Для виконання вміння або дії необхідно обрати ціль.
  • Використовуйте кнопки на екрані для активації вмінь або дій.
